# Sistema de Gestión de Inventario y Ventas

Proyecto de Base de Datos II.

## Descripción

Este proyecto consiste en el diseño y desarrollo de una base de datos NoSQL para una tienda electrónica.

El sistema permite gestionar productos, clientes, pedidos, stock e informes de ventas.

## Tecnologías utilizadas

- React
- Node.js
- Express
- MongoDB
- Mongoose

## Funcionalidades principales

- Gestión de productos
- Gestión de clientes
- Registro de pedidos
- Actualización automática de stock
- Informes de ventas por producto, categoría y mes
